# Notes on setting up neovim

- clone my neovim repo from github
- linux requirements (apt if available, snap if not):
  - `xclip` (for img-clip.nvim)
  - `python3`
  - `python3-pip`
  - `python3-venv`
  - `fzf`
  - `lazygit`
  - `ripgrep`
  - `clang`
  - `build-essential`

## Installing treesitter-cli:

```bash
sudo snap install rustup
rustup default stable
cargo install --locked tree-sitter-cli
```

## Kitty setup for fonts

Download the `Symbols Nerd Font`
Copy to `~/.local/share/fonts/`
